# 🦎 Newt Re-Identification (Re-ID) System

[![PyTorch](https://img.shields.io/badge/PyTorch-%23EE4C2C.svg?style=flat&logo=PyTorch&logoColor=white)](#)
[![Vision Transformer](https://img.shields.io/badge/Model-ViT_Base-blue)](#)
[![Accuracy](https://img.shields.io/badge/Rank--1-80.85%25-brightgreen)](#)
[![Accuracy](https://img.shields.io/badge/Rank--20-98.9%25-green)](#)

Проект представляет собой полностью автоматизированный ML-пайплайн для неинвазивной ре-идентификации (Re-ID) испанских иглистых тритонов (*Pleurodeles waltl*). Система позволяет распознавать индивидуальных особей по уникальному паттерну пятен на брюшке, избавляя экологов от необходимости использовать травмирующие методы кольцевания или чипирования.

## 🎯 Ключевые достижения
* **Высокая точность:** Модель уверенно находит правильного тритона с первой попытки в **80.85%** случаев (Rank-1).
* **Идеальный поиск:** В **98.9%** случаев правильная особь гарантированно находится в Топ-20 поисковой выдачи.
* **Автоматизация:** Полный цикл от сырой фотографии до выдачи ID особи без ручного выравнивания.

## 🧠 Архитектура решения

Пайплайн состоит из двух независимых нейросетей и векторной базы данных:

1. **Сегментация и Препроцессинг (Stage 1):**
   * Выделение брюшка животного на сыром фото с помощью модифицированной U-Net.
   * **Канонизация:** Автоматическое определение положения лап/головы и строгая вертикальная ориентация (90°) для унификации данных.
   * Удаление фона и конвертация в RGB (прозрачные зоны заменяются черным цветом) для концентрации внимания модели на узоре.

2. **Извлечение признаков / Backbone (Stage 2):**
   * Архитектура: **Vision Transformer (ViT-Base-Patch16)**.
   * Входное разрешение: **256x256 пикселей** (оптимизировано для сохранения мелких деталей пигментации).
   * Механизм *Self-Attention* позволяет сети анализировать не только отдельные пятна, но и их глобальное взаимное расположение (геометрию узора).

3. **Metric Learning:**
   * Функция потерь: **Batch Hard Triplet Loss** (Margin = 0.3).
   * Обучение: 50 эпох, оптимизатор `AdamW`, планировщик `CosineAnnealingLR`.
   * Использование эмбеддингов размерностью 512.

4. **Инференс:**
   * L2-нормализация выходных векторов.
   * Сравнение по метрике косинусного сходства (Cosine Similarity).

## 🛠 Установка и настройка

**1. Клонирование репозитория**
```bash
git clone <url-вашего-репозитория>
cd newt1_reid_project
```

**2. Установка зависимостей**
```bash
pip install -r requirements.txt
```

**3. Модели и веса**
Для работы системы необходимо загрузить веса моделей: https://drive.google.com/drive/folders/1dLLGGPGHIaPuN4iLunmSy20WT0Ev13D2

* **Model 1 (Re-ID):** Папку `models/` и `data/` поместить в корень проекта.
* **Model 2 (Segmentation):** Папку поместить в `test_model/models/`.

## 🚀 Использование

### Распознавание (Inference)
Для анализа фотографии и поиска тритона в базе данных запустите:
```bash
python inference.py data/gallery/1/IMG_9301.JPG
```
**Результат:** Вы получите JSON-ответ с ID лучшего совпадения, уверенностью (confidence) и списком Топ-20 кандидатов.

### Дополнительные инструменты
* `build_vector_db.py` — Пересборка базы признаков (выполнять при обновлении галереи или замене модели).
* `evaluate.py` — Расчет метрик (Rank-1, mAP) на тестовой выборке.
* `train.py` — Скрипт для дообучения модели.
